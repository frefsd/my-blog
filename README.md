# 🌟 我的个人博客 | Personal Blog

一个简洁美观、响应式设计的个人博客页面，用于展示个人简介、技能、成长时间线与联系方式。项目基于纯 HTML 和 CSS 构建，无框架依赖，轻量高效，适合作为个人主页或作品集入口。

🌐 **在线预览**: https://frefsd.github.io/my-blog/

## ✨ 特性 Features

- ✅ **响应式布局**：适配桌面、平板与手机端浏览
- 🎨 **现代 UI 设计**：渐变背景、毛玻璃导航、悬停动效
- 🧩 **模块化结构**：头像区、技能条、横向时间轴、联系卡片
- ⚡ **纯静态构建**：仅使用 HTML + CSS，无需后端或数据库
- 🔍 **语义化标签**：良好的 HTML 结构，利于 SEO 与可访问性
- 📁 **代码清晰**：CSS 模块化注释，易于维护与二次开发

---

## 🛠️ 技术栈 Tech Stack

- **HTML5** - 页面结构
- **CSS3** - 样式与动画（Flexbox、渐变、媒体查询）
- **Google Fonts / Segoe UI** - 字体渲染
- **GitHub Pages** - 免费托管部署
